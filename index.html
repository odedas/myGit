Hello Git

some basic commands to start working with git and github

use git init to start working with git

use git status to see whats going on

use git add [file] to include in what will be committed

use git commit -m "my first commit"to commit with -m (messsage to commit)

use git log to view what happened

        commit 8cc92de4b86024be7c66686522071bd1643a6949 (HEAD -> master)
        Author: Oded Assaraf 
            Date: Sat Apr 17 14:26:01 2021 +0300
            my first commit

add some cool style and commit again

use git checkout [commit id] to return to some point int timeline

use git branch to view your branches or git checkout -b feature1 to create new branch and switch to it

use git merge to merge your changes, you can use --squash flage to keep clean merge
